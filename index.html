FAQ Frequently Asked Questions Answered

What do I get in return for participating in the Token Generation Event (TGE)?

Upon successful Token Generation Event, you will receive GAMMA tokens in return for your investment. GAMMA tokens are used for governing the Gamma Options project, as well as profit accrual through burning mechanism.

What do I need to get started?

You'll need an Ethereum wallet that can interact with this webpage (web3-enabled wallet). We recommend using Metamask for both desktop and mobile. You can connect Metamask by clicking connect button, then selecting Metamask on desktop, or tapping connect then Wallet Connect on mobile.
